# Projeto integra: Inova Tech
[Projeto no portfólio!](https://app.cursar.me/giovanibraz/portfolio/inova-tech-projeto-em-conjunto-1772232418550)

## Introdução

Neste projeto, simulei junto com minha equipe a criação de uma empresa fictícia. O objetivo central foi vivenciar o fluxo real de desenvolvimento, desde a organização de pastas até o versionamento de código com GitHub Desktop, separando as responsabilidades entre Frontend e Backend.

## O Desafio

O desafio consistia em entregar uma presença digital completa para a empresa e um sistema interno de gerenciamento de estoque que seguisse a mesma lógica de produtos.

## Frontend

A equipe do frontend focou em criar um site institucional completo, priorizando a semântica e a responsividade.

- Tecnologias: HTML / CSS / JS
- Páginas desenvolvidas: Home, Sobre, Produtos e contato
- Destaques técnicos: Padronização, responsividade e performace

## Backend

No lado do servidor (simulado via terminal), desenvolvemos um sistema CRUD (Create, Read, Update, Delete) para o cadastro de produtos, utilizando um arquivo .txt para armazenar esses dados.

- Tecnologias: Python
- Arquivos: main.py, produtos.py e dados.txt
- Destaques técnicos: Persistência, Modularização e Validação de dados

## Colaboração e Fluxo Git

O ponto mais forte deste projeto foi o uso do Git e GitHub desktop. Seguimos padrões de mercado para garantir que o código fosse organizado.

- Commits Semânticos: feat, fix e style
- Trabalho em Branches: Ninguém mexeu direto na "main". Usamos branches para cada equipe e abrimos Pull Requests (PRs) para revisão.
- Resolução de Conflitos: Aprendemos a lidar com alterações simultâneas no mesmo repositório.

## O que aprendemos?

- A importância da comunicação entre times para manter o contexto do projeto.
- Como estruturar um projeto do zero de forma organizada.
- A base do Versionamento de Código, essencial para qualquer desenvolvedor profissional.
- Lógica de programação aplicada a problemas reais de cadastro e exibição de informações.
